ACESSO A UMA PAGINA WEB

O acesso é repleto de etapas desde a URL até a visualização do site. Mas basicamente, o navegador faz a solicitação para o servidor usando o protocolo (http ou https); o servidor responde com os dados e o navegador interpreta exibindo-os.

HTML É UMA LINGUAGEM DE PROGRAMAÇÃO?

HTML não é uma linguagem de programação geral como CSS, Java e JavaScript. É uma linguagem de marcação.

O QUE É CSS?

CSS é usado para a estilização. Por exemplo:

Criei uma classe "container" com uma classe "exemplo" dentro dela, e dentro da classe "exemplo" um circulo e um quadrado.

com o CSS eu ajustei a altura utilizando um "height" e a largura utilizando um "width". Para fazer o box apenas utilizei um border com cor preta.

ajustei as cores utilizando o "background-color.

ajustei as posiçoes utilizando o display flex

Para o alinhamento dos itens utilizei as especificaçoes do flex "justify content" e "align-items".

usei um space space-between para eles ficarem um em cada canto do box. Tambem usei um padding com 10 pixel para eles nao enconstarem nas bordas.
